# Footer

## Rôle
Pied de page du site : liens de contact (email, LinkedIn, CV PDF) et copyright.

## Props
Aucune prop — contenu statique.

## Tokens Lyse utilisés
- `--border` → bordure supérieure
- `--background` → fond
- `--text-secondary` → liens
- `--text-tertiary` → copyright

## Variants / états
Aucun état interactif hors hover sur les liens (transition `text-primary`).

## Décisions design
`id="contact"` permet d'ancrer le lien nav `#contact` sans section dédiée — le footer remplace une section contact (décision IA §4).
